INTRODUCTION

Dans ce cours, il sera question de parler du langage HTML. En effet, HTML est un langage assez simple, composé d'éléments applicables à des fragments de texte dans un document, pour leur donner un sens différent.

MODULE I : LE LANGAGE HTML

Dans ce module, on va définir le langage HTML. HyperText Markup Language (HTML) est le code utilisé pour structurer une page web et son contenu. A titre illustratif, le contenu de votre page pourra être structuré en un ensemble de paragraphes, une liste à puces ou des images et des tableaux de données.

TITRE 2 : LE CSS DANS HTML

Le CSS vient résoudre un problème bien différent du HTML : en effet, le HTML sert à définir les différents éléments d'une page, à leur donner du sens. Le CSS, lui, va servir à mettre en forme les différents contenus définis par le HTML en leur appliquant des styles.

TITRE I : PRINCIPAUX ELEMENENTS

Un élément HTML peut être soit constitué d'une paire de balises (ouvrante et fermante) et d'un contenu,soit d'une balise unique qu'on dit alors « orpheline ». L'élément p (qui sert à est par exemple constitué d'une balise ouvrante,d'une balise fermante et d'un contenu textuel entre les balises.

TITRE 3 : INDEX DES BALISES HTML

Lorsqu'un client HTTP demande une URL qui pointe vers une structure de répertoires au lieu d'une page Web réelle dans le répertoire, le serveur Web affichera généralement une page par défaut, souvent appelée "index".
